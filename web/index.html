SLD

Consulta: WS-Mapas/resources/sld?json=
id id único de la capa
filtro JSONArray de filtros, de estar vacio el arreglo se crea un sld generico
atributo Es el campo de la base de datos sobre el que se aplicará el filtro. Por ejmplo: se puede crear un filtro que sea coloreal todo los aeropuertos que sean de tipo internacional en una gama de verdes usando el atributo "objectid". Esto nos dara un mapa donde a cada uno de los "objectid" tendrá su propio color de la gama
